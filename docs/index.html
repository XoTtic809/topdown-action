v4.1.0 - SHOP REVAMP & ECONOMY UPDATE
NEW IMPROVED FIXED
  • 🏆 Champion Skins Section: Gold, Silver, and Bronze Champion skins now have their own dedicated section in the Skins tab with rank labels and gold-border styling. They are clearly marked as Leaderboard Exclusive and non-tradeable.
  • ⛔ Champion Trading Restriction: Champion skins are permanently blocked from the Marketplace at three independent security layers — client pre-check, server eligibility, and atomic Firestore transaction. No exploit can bypass all three.
  • 🛒 Shop Access — Legacy Accounts: Existing accounts created before this update now have immediate Marketplace access. New accounts registered after v4.1.0 still wait the standard 7 days.
  • 🎨 Oblivion Crate: Price updated to 15,000 coins. All 10 Oblivion skins now display correctly in the shop with unique dark gradient previews (previously appeared invisible).
  • 💰 Economy Rebalance: Enemy coin values increased (fast/basic: 1→2, tanks: 4→5, enforcers: 6→8, minibosses: 10→14). Wave clear bonus formula improved (was wave×2.5, now wave×4+15 so Wave 1 gives 19 coins instead of 2). Boss kill rewards increased ~25% at all tiers.
  • ⚡ XP Progression: Battle Pass XP rates buffed — perKill 5→6, perWave 22→26, match completion 35→50. Boss and elite kill XP multipliers improved. All 50 free track tier coin rewards increased 30–40%.
  • 🏪 Marketplace Pricing Fixed: Icon skins were incorrectly priced at 25,000 coin floor on the Marketplace (same as Mythic) despite coming from a 1,000 coin crate — fixed to 500–4,000 range. Duplicate `icon` key bug in RARITY_PRICING removed. Shop skin tier thresholds corrected so Diamond maps to Epic and Celestial Nexus to Mythic.
  • 🔧 Admin Panel: New Trade Restriction controls let admins block/unblock any skin from the Marketplace and view all active restrictions. New Skin Price Editor adjusts shop skin prices live without a deploy. New Crate Price Editor adjusts crate costs live. All changes are logged to Firestore for audit.

NEW IMPROVED FIXED
  • 🎫 BATTLE PASS SEASON 1: 50-tier progression system with free and premium tracks! Earn XP from kills (5 XP), waves (22 XP), and match completions (35 XP) with boss kill multipliers. Premium Pass available for 2,500 coins

  • 🌊 Wave Announcements: Large animated "WAVE #" banner with bounce-in animation at the start of each wave
  • 🌈 Cosmetics Tab: New shop tab for equipping trails and death effects earned from the Battle Pass
  • ☄️ Trail Effects (4 Types): Comet (cyan), Lightning (yellow), Flame (orange), and Void (purple) — particle trails that follow your player
  • 💀 Death Effects (2 Types): Starburst (100 rainbow particles) and Supernova (200 particles with shockwave) — triggered on player death

  • 🛡️ Anti-Cheat v2: Advanced score validation with plausibility checks, kill rate analysis, submission rate limiting, session tracking, and automatic bans for impossible scores
  • 🚫 Username Filter: Smart filtering with leet speak normalization, duplicate letter compression, and content filtering on signup
  • 🎯 FPS Counter: Toggleable FPS display in settings — color-coded green/yellow/red based on performance
  • 💾 Smart Save System: Intelligent save throttling with 30-second minimum intervals, 5-second batching, critical event priority (game over, purchases, level-ups), and automatic retry with exponential backoff on failure
  • 🔧 Admin Flagged Scores: Admin panel now shows flagged suspicious scores with severity levels, kill rates, and ban/dismiss actions
  • 🐛 Leaderboard Fix: Fixed leaderboard entries not rendering correctly due to CSS class mismatch — top 3 now properly highlighted with gold, silver, and bronze styling
  • ⚡ Performance: Particle cap (500 max) prevents lag on high waves, optimized Firebase reads with stats caching, admin pagination, and reduced redundant announcement fetches
